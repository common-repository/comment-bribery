== Description ==

Reward your readers who frequently comment with free gifts/prizes. You can define up to 5 different emails (including attachments) that are to be sent after a reader has commented a certain number of times.

== Installation ==
1a. Backup, backup then backup again...
1. Verify that you have PHP5, which is required for this plugin.
2. Upload the whole `comment-bribery` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. In WordPress admin go to 'Settings' -> 'Comment Bribery' and  configure as required
